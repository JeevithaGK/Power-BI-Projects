# HR Analytics Dashboard | Power BI

## Objective
To analyse employee data and understand workforce trends related to employee status, attrition, demographics, performance, and engagement to support HR decision-making.

## Dataset
- Source: HR Employee Dataset
- Includes employee status, department, age group, gender, division, performance score, engagement score, and training cost.

## Tools Used
- Power BI
- Power Query
- Microsoft Excel

## Key Analysis
- Active vs exited employees distribution
- Employee status by department, division, age group, and gender
- Employee joiners trend over time
- Performance score distribution by employee status
- Relationship between training cost and engagement score

## Dashboard Features
- KPI card showing total employees
- Donut charts for employee status and gender distribution
- Line chart showing employee joiners over time
- Bar charts for department, age group, and division analysis
- Scatter chart comparing training cost and engagement score
